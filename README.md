# 🛒 Retail Revenue Optimization using Sales & Inventory Analysis

## 📌 Project Overview

This project focuses on analyzing retail sales and inventory data to identify revenue trends, profitability issues, customer purchasing behavior, and regional sales performance using Python, MySQL, SQL, and Power BI.

The goal of this project is to help retail businesses make data-driven decisions and improve overall business performance through analytics and visualization.

---

# ❗ Problem Statement

Retail businesses often face several operational and financial challenges, including:

- Unsold or slow-moving inventory
- High discounts reducing profitability
- Lack of visibility into top-performing products
- Uneven regional sales performance
- Difficulty tracking customer purchasing trends
- No centralized dashboard for monitoring business KPIs

These issues can lead to revenue leakage, poor inventory decisions, and reduced profit margins.

---

# 💡 Solution

To solve these business problems, an end-to-end retail analytics solution was developed using Python, MySQL, SQL, and Power BI.

The solution includes:

## ✅ Data Cleaning & Preprocessing
- Cleaned raw retail data using Python
- Handled missing values and duplicates
- Converted date columns and standardized data formats

## ✅ SQL-Based Business Analysis
Performed SQL queries to analyze:
- Sales by category
- Profit by category
- Monthly sales trends
- Region-wise sales performance
- Top-selling products
- Worst-performing products
- Loss-making products
- Customer purchasing behavior

## ✅ Interactive Power BI Dashboard
Built a business dashboard with:
- KPI cards for Revenue, Profit, Orders, and Average Order Value
- Sales trend analysis
- Product performance tracking
- Regional sales insights
- Customer insights and business recommendations

---

# 📊 Dashboard Features

- Total Revenue KPI
- Total Profit KPI
- Average Order Value KPI
- Total Orders KPI
- Category-wise Sales Analysis
- Region-wise Sales Analysis
- Monthly Sales Trend
- Top 10 Selling Products
- Worst Performing Products
- Loss-Making Product Analysis

---

# 🔍 Key Insights

- Technology category generated the highest revenue.
- West region showed the strongest sales performance.
- Several products generated losses despite high discounts.
- Sales increased significantly during year-end periods.
- Some low-performing products negatively affected profitability.

---

# 📈 Business Impact

This project helps businesses:
- Improve decision-making using analytics
- Reduce loss-making inventory
- Optimize discount strategies
- Identify high-performing products and regions
- Monitor overall business performance efficiently
